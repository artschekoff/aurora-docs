# Поиск с автокомплитом

> [← Сводный план модулей](../MODULES.md) | Источник: [REQS.md](../REQS.md) §2 (критичные, #4) | [TZ.md](../TZ.md) §1.4

## Статус: 🔌 Нужен новый плагин + кастомная разработка (поиск по фото)

**Рекомендуемые плагины:**

- **FiboSearch (AJAX Search for WooCommerce)** (Freemium) — автокомплит с фото и ценами
- **SearchWP** (платный) — мощный поиск
- **Relevanssi Premium** — расширенный поиск

## Основное требование

Поиск с автокомплитом: при вводе текста всплывают до 7 подсказок с фото товара и ценами. Поиск по наименованию изделия, бренда. Дополнительно — поиск по фото (загрузка изображения).

## Что нужно сделать

### Текстовый поиск (плагин)

1. **Установить FiboSearch** (Pro предпочтительно)
2. **Настроить автокомплит** — фото товара + цена в подсказках, max 7 штук
3. **Настроить индексацию** по наименованию, бренду, коллекции

### Поиск по фото (кастомная разработка)

1. **Интеграция с внешним API** визуального поиска (Google Vision API, Clarifai или кастомная ML-модель)
2. **Кастомный модуль** загрузки фото в поисковую строку
3. **Матчинг результатов** с каталогом WooCommerce

> Подробнее по поиску по фото: [PHOTO-SEARCH.md](PHOTO-SEARCH.md)
