# Система Аватар / кастомизация

> [← Сводный план модулей](../MODULES.md) | Источник: [REQS.md](../REQS.md) §2 (кастом, #19) | [TZ.md](../TZ.md) §3.1, 5.1.2

## Статус: 🛠️ Кастомная разработка

**База:** ACF PRO (кастомные поля пользователя)

## Основное требование

Уникальная система кастомизации изделий под клиента: создание личного Аватара с размерами, метриками и предпочтениями (цвет, стиль). Данные Аватара используются при заказе изделий и для персональных подборок из будущих коллекций.

## Покрытие

| Требование | Раздел ТЗ | Покрытие |
|------------|-----------|----------|
| Кастомные поля Аватара (размеры, метрики) | 5.1.2 | 🔧 ACF PRO — поля есть, нужна логика |
| Страница анкеты в ЛК | 5.1.2 | 🛠️ Кастомная страница |
| Автоподгрузка данных Аватара в карточку товара | 4.2 | 🛠️ Кастомная логика |
| Визуал Аватара с метриками при регистрации | 3.1 | 🛠️ Кастомная разработка |
| Персональные подборки на основе предпочтений | 5.1.2 | 🛠️ Кастомная логика рекомендаций |

## Что нужно разработать

1. **Группа полей ACF PRO** для профиля пользователя:
   - Размеры (грудь, талия, бёдра, рост)
   - Размер одежды (RU/EU/US)
   - Предпочтения по цвету, стилю
   - Знак зодиака, психотип (связь с [ASTROLOGY.md](ASTROLOGY.md))
2. **Кастомная страница анкеты** в ЛК — пошаговая форма заполнения с визуализацией
3. **Визуал Аватара** — графическое представление метрик при регистрации (попап с картинкой + метками)
4. **Логика автоподгрузки** — при просмотре карточки товара подставлять размер клиента автоматически
5. **Рекомендательная логика** — подборки товаров на основе предпочтений Аватара

**Оценка сложности:** Высокая

## План работ и оценка

1. Проектирование состава полей, сценариев анкеты и связей с каталогом — 6 ч
2. Настройка ACF-структуры профиля пользователя — 6 ч
3. Разработка пошаговой анкеты в личном кабинете — 12 ч
4. Реализация визуала Аватара и отображения метрик — 8 ч
5. Интеграция автоподстановки данных в карточку товара и checkout — 8 ч
6. Разработка логики персональных подборок и тестирование — 8 ч

**Итого:** 48 ч
