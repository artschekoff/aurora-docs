# Поиск по фото

> [← Сводный план модулей](../MODULES.md) | Источник: [REQS.md](../REQS.md) §2 (кастом, #22) | [TZ.md](../TZ.md) §1.4

## Статус: 🛠️ Полная кастомная разработка

**Нет готового плагина.** Требуется интеграция с внешним API.

## Основное требование

Возможность загрузить фотографию в поисковую строку и найти визуально похожие товары в каталоге.

## Покрытие

| Требование | Раздел ТЗ | Покрытие |
|------------|-----------|----------|
| Поиск товаров по загруженному фото | 1.4 | 🛠️ Полная кастомная разработка |

## Что нужно разработать

1. **Кнопка загрузки фото** в поисковой строке (рядом с текстовым поиском)
2. **Интеграция с API визуального поиска:**
   - Google Cloud Vision API
   - Clarifai
   - или кастомная ML-модель
3. **Модуль матчинга** — сопоставление результатов API с товарами в каталоге WooCommerce
4. **Обработка изображений** — ресайз, оптимизация перед отправкой в API
5. **UX** — показ результатов в формате каталога с релевантностью

**Оценка сложности:** Высокая

## План работ и оценка

1. Исследование и выбор API визуального поиска — 4 ч
2. Проектирование пайплайна загрузки и обработки изображений — 6 ч
3. Реализация интеграции с внешним API — 10 ч
4. Разработка модуля матчинга с товарами WooCommerce — 10 ч
5. Реализация UI результатов и UX-обработки ошибок — 4 ч
6. Тестирование качества поиска и калибровка — 6 ч

**Итого:** 40 ч
