# Email (транзакционная отправка)

> [← Сводный план модулей](../MODULES.md) | Источник: [REQS.md](../REQS.md) §1.5 | [TZ.md](../TZ.md) §6.4

## Статус: ✅ Полностью покрыто плагинами

**Плагины:**

- WP Mail SMTP 4.7.1 (установлен) — транспорт
- WooCommerce 10.3.8 (установлен) — генерация писем по событиям

## Основное требование

Надёжная доставка транзакционных писем: регистрация, восстановление пароля, смена статуса заказа. Отправка через SMTP/API для гарантированной доставки.

Не требуется для транзакционных писем. Для маркетинговых рассылок см. [EMAIL-MARKETING.md](EMAIL-MARKETING.md).
