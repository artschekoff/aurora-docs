# Мультиязычность

## Статус: ✅ Полностью покрыто плагинами

**Плагины:**
- Polylang Pro 3.5.1 (установлен)
- Polylang for WooCommerce 1.6.3 (установлен)

## Основное требование

Мультиязычный сайт на 3 языках (RU / EN / CN) с уникальным контентом для каждого региона. Переключение языка = переход на отдельную страницу с отдельным контентом. Язык админки настраивается индивидуально для каждого пользователя.

## Доработка

Не требуется. Polylang Pro полностью покрывает все требования по мультиязычности.
